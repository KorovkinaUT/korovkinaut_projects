## Содержание репозитория

### 📁 [Link Tracker](./LinkTracker)
Telegram-бот для отслеживания обновлений на сайтах с уведомлениями в Telegram. Написан на **Go** и содержит:
- **Telegram-бот** — интерфейс для пользователя (команды `/track`, `/untrack`, `/list`)
- **Scrapper** — периодически проверяет ссылки и отправляет обновления
- **База данных** PostgreSQL для хранения данных
- **Unit**-тесты и **интеграционные** тесты

### 📁 [Raytracer](./Raytracer)
Трассировщик лучей на C++. Программа читает 3D-сцены в формате `.obj` (с материалами `.mtl`), выполняет рендеринг с использованием модели освещения Фонга и поддерживает:
- Отражение и преломление света
- Мягкие тени
- Настраиваемые материалы (Ka, Kd, Ks, Ns, Ni)
- Режимы отладки: глубина, нормали
- Пост-обработка: тонмаппинг, гамма-коррекция

### 📁 [База данных для больницы](./HospitalDatabase)
Реляционная база данных для больницы, содержит:
- **models/** — концептуальная, логическая и физическая модели БД
- **requests/** — примеры SQL-запросов

### 📁 [Hunt The Wumpus](./HuntTheWumpusGame)
Консольная текстовая игра — вариация классической "Hunt the Wumpus" на **python**:
- Игрок исследует пещеры (граф-додекаэдр), избегает ловушек (ямы, летучие мыши) и охотится на Вампусов
- Настраиваемые параметры: количество Вампусов, летучих мышей
- Стрельба с поворотом стрелы
- Система ощущений (ветер, шум, запах)
- Рейтинг в звёздах по итогам игры

### 📁 [Алгоритмы](./Algorithms)
Классические алгоритмы и структуры данных на **C++**:
- Умножение полиномов через **FFT**
- **AVL-дерево** (поиск, вставка, lower_bound)
- Поиск **мостов** в графе и **компонент сильной связности** (Косарайю)
- **Выпуклая оболочка** (алгоритм Джарвиса)
- **Задача о рюкзаке** (ДП с восстановлением ответа)
- **Куча** (min-heap) и **min-max куча**
- **Быстрая сортировка**
- **Sparse Table** для RMQ2 (вторая порядковая статистика)
- **Trie** для поиска пар строк, конкатенация которых — палиндром

### 📁 [Структуры на С++](./C++_Containers)
Реализация стандартных контейнеров и утилит из STL «с нуля» на C++:
- `BigInteger` — целые числа произвольной длины
- `Rational` — рациональные числа
- `String` — строка с динамическим буфером
- `Deque` — двусторонняя очередь с итераторами
- `List` со `StackAllocator` — аллокатор на стеке
- `SharedPtr` / `WeakPtr` — умные указатели с подсчётом ссылок
- `UnorderedMap` — хэш-таблица с цепочками
- `Geometry` — геометрические примитивы с наследованием и виртуальными функциями

### 📁 [АКОС](./CAOS)
Низкоуровневые программы на **C** и **ассемблере**:
- `utf8.c` — чтение/запись UTF-8 через системные вызовы
- `rm.c` / `mkdir.c` — аналоги консольных утилит
- `dot_product.s` — скалярное произведение на SSE (x86-64)

- `filewatcher.c` — мониторинг `write` через `ptrace`
